# ShellScriptsLibrary
Library of usefull shell scripts

## List of available scripts
- mp3-to-opus.sh
  - Convert existing MP3 files to OPUS files via FFmpeg
  - usage: ./mp3-to-opus INPUT_DIR_PATH OUTPUT_DIR_PATH

- prepend-idx.sh
  - Prepend index to files (and directories) in input directory
  - usage: ./prepend-idx.sh DIR_PATH
